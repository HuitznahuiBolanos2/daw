Cuestionario:


¿Qué es entonces una base de datos - BD?

Una base de datos es una serie de archivos interrelacionados los cuales deben de estar cuidadosamente diseñados para evitar la repeticion y que puedan cumplir con las necesidades de la empresa.



¿En qué cases es conveniente usar una base de datos?

En toda situacion que requiera del maneho de informacion y archivos y es requeriada por varios usuarios o aplicaciones.



¿Qué es un sistema de gestion de Base de Datos - DBMS?

Es una base de datos con un conjunto de proramas para acceder a dichos datos. El objetivo de esta es hacer un entorno conveniente y eficiente al momento de trabajar con los datos.



Enlista y Justifica las funciones/responsabilidades que tienen los DBMS:

Guardar incofmacion : El sistema debe ser capas de guardar la informacion importante de manera permanente.

Manejo de Informacion: El sistema debe de permitir a los usuarios manipular la informacion de manera efectiva, permitiendoles borrar, cambiar o actualizar informacion.

Indice de datos: El sistemma debe de tener un catalogo que clasifique y describa brevemente que datos posee.

Manejo de Seguridad: El sistema debe protejer la base de datos, autorizando a ususarios a accesar a la informacion que requieran solamente.

Respaldo de informacion: El sistema debe de proporcionar un mecanismo para respaldar la informacion de manera periodica, para evitar que esta se pierda.

Acceso Concurrente: El sistema debe permitir que dos personas accedan a la base de datos al mismo tiempo.

Acceso a interfaces de programacion: El sistema debe ser capaz de ser actualizado y controlado a traves de una interfaz de programacion.



Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En el proyecto del desarrollo de un videojuego, porque al ser un trabajo en equipo habia momentos que necesitabamos tener acceso a un recurso en especifico varias personas, pero al no poder tener dicho acesso el trabajo se entorpecio, si hubieramos tenido una base de datos sencilla a la hora de realizar el proyecto la dificultad de este habria disminuido considerablemente.
